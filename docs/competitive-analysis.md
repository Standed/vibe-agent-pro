## 二、各竞品详细分析

### 2.1 MulanAI

**Slogan**：Make videos, make changes

**核心功能**：
- 文本拆分节点（支持 Markdown）
- 模板化工作流，一键生成
- 节点可复制粘贴，团队共享
- 角色替换、动作复刻、数字人、对口型等专属节点
- 提示词注入（同一风格多次复用）
- Agent能力强大，类似 Claude Code，操作可追溯可视化

**产品亮点**：
| 功能 | 说明 |
|------|------|
| TodoList | Agent任务进度可视化 |
| 操作追溯 | 每步操作有记录 |
| 一键同款 | 用户可一键复制工作流 |
| 节点化 | 灵活组合，泛用性强 |

**适用场景**：动画短片、多场景视频批量生产

**借鉴点**：
- ✅ Agent TodoList 可视化
- ✅ 提示词注入复用机制
- ✅ 工作流模板化

---

### 2.2 OiiOii

**定位**：全球首个动画创作 Agent

**核心功能**：
- 托管模式引导：动画类型 → 影片长度 → 比例 → 对白语言 → 情绪关键词
- 多种模板：快速视频、音乐概念短片、剧情故事短片、漫画转视频、衍生品设计
- 对话批量修改分镜
- 手动裁剪与增加分镜
- 全局角色库

**产品亮点**：
| 功能 | 说明 |
|------|------|
| 用户引导 | 一步步确定参数 |
| 批量修改 | 对话式批量操作分镜 |
| 可爱风格 | 二次元动画词汇 |
| 反馈收集 | 科学决策，方便定价 |

**借鉴点**：
- ✅ 托管模式引导流程
- ✅ 全局角色库
- ✅ 批量修改分镜

---

### 2.3 Tapnow

**教程**：https://docs.tapnow.ai/zh/docs/huan-ying-lai-dao-tap/kuai-su-kai-shi

**核心功能**：
- 无限画布连续创作
- 分组功能、首尾帧功能
- 一次抽卡生成多次
- 一键拉片、分镜策划模板
- 节点拖拽到对话框（上下文引用）

**产品亮点**：
| 功能 | 说明 |
|------|------|
| 连续创作 | 从角色设计到分镜场景视频 |
| 节点拖拽 | 图片/视频拖到对话框生成提示词 |
| 一键拉片 | 分析视频生成分镜脚本 |
| 多次抽卡 | 一次生成多个结果 |

**劣势**：
- ❌ 内容过长时不便整体查看

**借鉴点**：
- ✅ 节点拖拽到对话框
- ✅ 一键拉片功能
- ✅ 首尾帧功能

---

### 2.4 Seko 2.0

**文章**：商汤Seko2.0来了，做AI漫剧又有了一个利器

**核心架构**：
- 三视图切换：策划环节 / 故事板视图 / 剪辑视图
- 首页设计感强，左侧小标签小巧

**产品亮点**：
| 功能 | 说明 |
|------|------|
| 美感UI | iOS液态玻璃、高斯模糊、圆角风格 |
| 一键生成 | 用户友好 |
| 分镜优化 | 点击分镜后可与AI对话优化 |
| 画布编辑 | 支持画布内编辑 |

**劣势**：
- ❌ 没有参考生视频整套逻辑
- ❌ 画布内没有Agent功能

**借鉴点**：
- ✅ 首页左侧小图标栏设计
- ✅ 三视图切换架构
- ✅ iOS液态玻璃美感

---

### 2.5 Medeo

**核心功能**：

**借鉴点**：
- ✅ 对话式 + TodoList

- ✅ 音乐节拍匹配

- 故事板视图

- 右侧 AI 助手

**产品亮点**：
- 模型丰富
- 首页可上传想法

**借鉴点**：
- ✅ 故事板 + 时间线经典布局

### 2.7 纳米AI

**核心特色**：
- 整体是剪辑视图

**借鉴点**：
- ✅ 分镜级无限画布
- ✅ 剪辑视图主导

### 2.8 Aipai

**核心架构**：
- **标准模式**：精细化调整

**产品亮点**：

**借鉴点**：
- ✅ 双模式分离
- ✅ 批量参考生视频

## 四、我们的差异化定位

### 4.1 定位

**Video Agent Pro = 中长视频创作平台**

| 维度 | 竞品主流 | 我们 |
|------|----------|------|
| 目标用户 | 短视频/探索型 | 中长视频/叙事型 |
| 核心架构 | 无限画布 | 场景结构化画布 + 草稿探索 |
| 对话模式 | 单一 | Agent批量 + Pro精调 |
| 角色一致性 | 单一方案 | 三视图(即梦) + Sora参考 双轨 |
| 数据层级 | 扁平 | Series → Project → Scene → Shot |

### 4.2 竞争优势

| 优势 | 说明 |
|------|------|
| **结构化管理** | 四级层次（剧集→项目→场景→分镜），适合长视频 |
| **双模式对话** | Agent批量 + Pro精调，效率与精度兼顾 |
| **角色一致性** | 三视图(即梦) + Sora参考 双轨方案 |
| **参考生视频** | Sora 2 原生支持，保持角色一致性 |
| **草稿探索** | 自由画布探索，结果可应用到分镜 |

### 4.3 需要补齐的能力

| 能力 | 优先级 | 参考竞品 |
|------|--------|----------|
| 一键拉片 | P2 | Tapnow |
| 模板系统 | P2 | MulanAI / OiiOii |
| 拖拽到对话 | P1 | Tapnow |
| Agent TodoList | P2 | MulanAI / Medeo |
| 音乐节拍匹配 | P3 | Medeo |

## 五、总结

### 5.1 竞品格局

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   MulanAI ●──────────────────────────────────────● Tapnow      │
│   (节点工作流)                                    (无限画布)   │
│        │                                              │        │
│        │                                              │        │
│        │      ┌─────────────────────┐                │        │
│        │      │   Video Agent Pro   │                │        │
│        │      │   (结构化 + 草稿)    │                │        │
│        │      └─────────────────────┘                │        │
│        │                                              │        │
│   Medeo ●──────────────────────────────────────● Seko 2.0     │
│   (对话驱动)                                    (美感UI)       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 我们的策略

1. **差异化**：场景结构化画布 + 草稿探索，解决长视频管理痛点
2. **双模式**：Agent批量 + Pro精调，效率与精度兼顾
3. **美感UI**：借鉴 Seko 2.0 的 iOS 液态玻璃风格
4. **拖拽交互**：借鉴 Tapnow 的节点拖拽到对话
5. **角色一致**：三视图 + Sora参考 双轨方案领先

### 5.3 核心竞争壁垒

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  四级结构（Series→Project→Scene→Shot）                     │
│           +                                                 │

│           +                                                 │
│  角色一致性双轨（三视图 + Sora参考）                        │
│           +                                                 │
│  草稿探索画布（自由创作 → 结构化成果）                      │
│                                                             │

│                                                             │
```
